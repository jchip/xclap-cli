# xclap CLI

CLI invoker for the [xclap] Javascript task executor and build tool.

# Install

```bash
npm install -g xclap-cli
```

# Usage

```bash
$ clap [options] <task> [task_options] <task> [task_options]
```

ie:

```bash
$ clap build
```
